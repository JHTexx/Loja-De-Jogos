
# Projeto: Loja de Jogos

## Descrição
Este sistema simula a gestão de uma loja de jogos, permitindo o cadastro, venda e controle de clientes, jogos e vendas. O projeto segue os princípios da programação orientada a objetos e a arquitetura MVC.

## Requisitos Atendidos
- ✅ 3 CRUDs: Cliente, Jogo, Venda
- ✅ Relações entre entidades (Venda possui Cliente e lista de Jogos)
- ✅ Herança (superclasse ModeloBase)
- ✅ Classe abstrata com método abstrato (`exibirDetalhes`)
- ✅ Polimorfismo de sobrescrita e sobrecarga
- ✅ Interface `CRUD<T>`
- ✅ Encapsulamento com atributos privados e métodos públicos
- ✅ Padrão MVC (Model, View, Controller)
- ✅ Laços e estruturas de controle no menu interativo
- ✅ Log em arquivo `.log`
- ✅ Serialização e desserialização de objetos
- ✅ Clean Code aplicado (nomes claros, modularidade, organização)

## Como Executar

1. Importe o projeto em uma IDE como **Eclipse** ou **IntelliJ**.
2. Compile e execute a classe `Main.java` (localizada em `src/Main.java`).
3. O menu será exibido no console para interação com o sistema.

## Observações
- O log do sistema é salvo no arquivo `loja_jogos.log`.
- Os dados são persistidos via serialização em arquivos `.ser`.
- O sistema é 100% orientado a objetos, com reutilização de código via herança e interface.
